# Projet_hypertension

## Description
Dans ce projet, on classifie les patients selon qu'ils sont atteints de l'hypertension ou non.

# Objectif
Réussir à comprendre les données et créer une première stratégie d'analyse.
On va chercher un recall(plus de faux négatifs pour éviter de manquer certains patients.

# Etapes réalisées
1. Analyse de forme
2. Analyse de fond
3. Analyse plus détaillée
4. Preprocessing
5. création d'un modèle de machine learning

# Technologies utilisées
1. Python
2. Pandas
3. Scikit-learn
4. Jupyter Notebook
